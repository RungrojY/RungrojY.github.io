1 Basic Usage

1.1 Start shell image

multipass shell <image-name>

1.2 Create virtual machine with yaml

Create Ubuntu Server image with yaml file. The word focal refers to Ubuntu 20.04 LTS.

multipass launch --name hadoop0 --network Wi-Fi --cloud-init setup-ubuntu.yaml focal
multipass launch --name hadoop1 --network Wi-Fi --cloud-init setup-ubuntu.yaml focal
multipass launch --name hadoop2 --network Wi-Fi --cloud-init setup-ubuntu.yaml focal
package_update: true
package_upgrade: true
packages:
  - openjdk-8-jdk

1.3 Delete images

Delete created images

multipass delete <image-name>

1.4 Transfer a file

Suppose I has a text file (test.txt) and I want to transfer a file from a Window system into Ubuntu VM

multipass transfer test.txt hadoop0:/home/ubuntu/
# or
multipass transfer C:\Users\Jorgnur\Downloads\hadoop-3.3.5.tar.gz hadoop0:/home/ubuntu/

Suppose I has a folder (test) and I want to transfer a file from a Window system into Ubuntu VM

multipass transfer -r test hadoop0:/home/ubuntu/

1.5 Set User Password

If you do not set the password at first when creating VM, use the following command to set new password.

sudo passwd ubuntu

1.6 Set a fixed IP address for your instances

The IP address assigned to instances is dynamic by default, to assign the IP address use this command

multipass set instance-name --address 192.168.1.22
#
multipass set instance-name --address 192.168.1.22

1.7 Add more resource

Stop the instance you want to modify:

multipass stop hadoop0

Increase the number of CPU cores:

multipass set hadoop0 --cpus 2

Increase the amount of RAM:

multipass set hadoop0 --mem 2G

Increase the disk size:

multipass set hadoop0 --disk 10G

Start the instance again:

multipass start hadoop0
